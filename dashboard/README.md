# 🤖 Team Dashboard - Управление AI агентами

Система для управления командой AI агентов разработки.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd dashboard
npm install
```

### 2. Настройка Telegram бота
1. Получите токен у @BotFather
2. Создайте файл `.env`:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

### 3. Запуск системы
```bash
# Запуск Dashboard (веб-интерфейс)
npm start

# Запуск Telegram бота (в другом терминале)
npm run bot

# Или оба вместе
npm run dev
```

## 🌐 Доступ

### Dashboard (веб-интерфейс)
- **URL:** http://localhost:3000
- **Функции:** Просмотр агентов, создание задач, управление

### Telegram бот
- **Команды:**
  - `/start` - Начало работы
  - `/tasks` - Список задач
  - `/new [задача]` - Создать задачу
  - `/status [агент]` - Статус агента
  - `/report` - Отчет за день
  - `/help` - Помощь

## 🏗️ Архитектура

```
Telegram Бот (управление) ↔ Dashboard API ↔ Агенты
         ↓                           ↓
   Команды пользователя      Визуализация состояния
```

## 👥 Агенты (по умолчанию)

1. **Design Architect** - Дизайн и прототипирование
2. **Fullstack Builder** - Разработка
3. **Deploy & SEO** - Деплой и оптимизация

## 📊 API endpoints

- `GET /api/agents` - Список агентов
- `GET /api/tasks` - Список задач
- `POST /api/tasks` - Создать задачу
- `POST /api/assign` - Назначить задачу агенту

## 🎯 Примеры использования

### Создание задачи через Telegram:
```
/new Создать лендинг для канала
Приоритет: высокий
Дедлайн: 2026-03-17
Описание: Нужен продающий лендинг для канала AI Бизнес Автоматизация
```

### Проверка статуса:
```
/status Design Architect
/status все
```

## 🔧 Расширение системы

### Добавление новых агентов:
1. Отредактируйте `app.js` (массив `agents`)
2. Добавьте навыки и специализацию

### Интеграция с реальными агентами:
1. Настройте webhook для уведомлений
2. Добавьте логику назначения задач
3. Интегрируйте с MCP (Model Context Protocol)
